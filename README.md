# Innova Acabados Global 🚗

Sitio web oficial de **Innova Acabados Global** — Especialistas en carrocería y electrónica automotriz.

## 🛠️ Servicios
- Planchado Automotriz
- Pintura de Alta Gama
- Diagnóstico Electrónico
- Reset Sistema Airbag
- Simulaciones Electrónicas
- Calibración de Odómetros

## 🚀 Tech Stack
- **React 18** + **Vite 5**
- **Tailwind CSS 3**
- **Framer Motion** — Animaciones suaves
- **Lucide React** — Iconografía
- **React Icons** — Íconos de redes sociales
- **React Scroll** — Scroll suave entre secciones
- **React CountUp** — Contadores animados

## 📦 Instalación

```bash
# Clonar repositorio
git clone https://github.com/tu-usuario/innova-acabados-global.git

# Instalar dependencias
cd innova-acabados-global
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## 🏗️ Build para producción

```bash
npm run build
```

## 🌐 Deploy con Vercel

Este proyecto está configurado para deploy automático con Vercel.

1. Sube el repositorio a GitHub
2. Importa el proyecto en [vercel.com](https://vercel.com)
3. Vercel detecta automáticamente la configuración de Vite
4. ¡Listo! Deploy automático en cada push
